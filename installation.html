Installation of NuPyCEE

Don't be worried, NuPyCEE does not need to be installed. Just download it.

Download code

All NuPyCEE is publicly available on github. To download the code git is required on your system. Now download a copy into your local directory via 

         git clone https://github.com/NuGrid/NuPyCEE.git

The repository can be found here.

Dependencies

All python dependencies are listed in install_dependencies.sh available in the root directory of NuPyCEE. You might be able to execute it with your shell.
