Web interface instructions

The web interface let the user calculate (multiple) stellar populations which can be analyzed and ejecta dowloaded. SYGMA allows to follow the ejecta of of simple stellar populations in a closed box. OMEGA deals with the simulation of galaxies with inflows and outflows.
